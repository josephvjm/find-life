# Find Life

Open-source phylogenetic data infrastructure providing offline-capable access to authoritative taxonomic and evolutionary relationship data.

## Status

Currently in development. Public release planned for **October 1, 2026**.

## About

Find Life provides researchers, developers, and institutions with reliable phylogenetic data through:

- **Python library** for programmatic access to taxonomic relationships
- **Command-line interface** for batch processing and field research
- **Offline capability** using locally-cached ITIS database
- **Clean, well-documented API** built on authoritative ITIS data

## Reference Implementation

See the Find Life API in action at **[find-life.org](https://find-life.org)** — a free educational tool demonstrating how reliable phylogenetic infrastructure enables accessible biodiversity exploration.

## Development

This project is under active development. Public contributions will be welcomed after the initial v1.0 release in October 2026.

Follow progress through bi-weekly demo videos and commit history.
